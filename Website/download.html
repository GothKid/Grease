Installation Instructions

  1. Download the release you wish to use. It is recommended that you always choose the current version, though there is no support provided for any version.
  2. Unzip the file
  3. Open the new folder and click on "Setup.exe"
  4. If you have earlier versions of Grease, you may need to uninstall before you attempt upgrading.
